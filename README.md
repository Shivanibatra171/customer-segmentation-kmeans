# 🛍️ Customer Segmentation

A unsupervised machine learning project that segments mall customers based on spending behavior.

## 🔍 Problem Statement
Group mall customers into distinct segments based on their annual income and spending score to help businesses target the right audience.

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn (KMeans Clustering)
- Matplotlib, Seaborn

## 📊 Dataset
- Mall Customers dataset
- Features: CustomerID, Gender, Age, Annual Income, Spending Score

## ⚙️ Approach
1. Exploratory Data Analysis
2. Feature selection
3. KMeans clustering
4. Elbow method to find optimal clusters
5. Visualization of customer segments
